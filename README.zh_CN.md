# IDA-DataExportPlus
一个用于导出数据的IDA PRO插件，它可以替换IDA Pro原始的数据导出窗口。
该插件支持基于不同数据类型的数据导出，包括不同大小的整数和浮点类型。除此之外也支持导出汇编代码。

## 安装
复制`dataexportplus.py`到IDA插件目录下。需要IDA Pro版本7.6或以上

## 使用方法
使用快捷键`Shift+E`以唤出插件窗口
